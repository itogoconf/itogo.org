La première édition de la conférence ITogo aura lieu le samedi 9 juin 2018 à l'Agora Senghor près du Collège Saint Joseph de Lomé. Inscription sur Meetup

Les talks

Pour cette première édition vous retrouverez les sessions suivantes

50 minutes pour créer une application Android

09:00-09:50

Le but de ce tool in action est de vous montrer en 50 minutes comment créer une application Android. Nous ferons un retour sur les concepts de la prgrammation Android et nous développerons en Kotlin, le nouveau langage de référence pour développer sous Android. Kotlin a redonné un réel coup de jeune aux développements sur cette plateforme. A la fin vous devrez avoir les informations de base pour créer vos propres applications.

par Guillaume EHRET

JSR et JDK hacking

10:00-10:50

Durant cette présentation nous découvrirons les standards de Java à travers un parcours des objectifs et fonctionnement de la JCP ( Java Community Process). Nous allons consacrer quelques minutes pour procéder à la compilation et génération de l’exécutable du JDK ( kit de développement de Java) à partir de la source

par Constantin DRABO

Les logiciels libres : que m'apporte t'il en tant qu'africain, togolais?

11:00-11:50

Le but de cette présentation est de partager mon regard sur l'utilisation des logiciels libres au Togo. Cette présentation va permettre d'expliquer le concept de logiciel libre et et son impact socio-économique au Togo. On parlera des logiciels libres d'un point de vue utilisateur normal , développeur, entreprise et gouvernement en passant par ses avantages et cas de figures.

par Josue TCHIRKTEMA

Optimiser les performances d'une application web

14:00-14:50

Avec les outils actuels, développer une application web (grand public ou privée), avec du JS en front et du Java en backend est de plus en plus rapide. Les premiers hics peuvent survenir le jour où vous basculez cette application en production. Mon but est de vous montrer dans cette session, comment optimiser une webapp quelque soit les solutions techniques utilisées côté front ou côté serveur. Mes exemples se basent sur une application SpringBoot avec un frontend en JS (sans framework particulier) mais ceci est peu important. Nous allons nous focaliser sur les concepts qui restent vrais quelque soit les langages.

Nous verrons les différentes étapes pour améliorer la rapidité d'une application au premier chargement (load time), en mode offline, comment jouer sur le critical rendering... Au niveau des solutions nous parlerons cache, asynchronisme, services workers... Par cette session j'aimerais partager ce que j'ai appris en créant des sites web publics et notamment celui de la conférence Mix-IT, les erreurs à ne pas faire, comment améliorer les performances (ce que nous avons fait, ce nous pouvons encore faire pour nous améliorer)

par Guillaume EHRET

Mes Premiers pas sur Hadoop

15:00-15:50

Le but de cette présentation est de partager mes premières impressions sur Hadoop. C’est une présentation qui parlera d'Hadoop du point de vue d’un développeur lambda qui arrive dans l'écosystème Hadoop. Hadoop par où commencer ? Quels sont les avantages de la plateforme et l’écosystème. Ce sera une présentation accessible qui parlera d'Hadoop pas du point de vue des grands géants du web ( Facebook, Yahoo, Google…) mais plutôt pour nous dans nos entreprises (à taille humaine). Comment Hadoop peut s'intègre dans un système d'information. Qu'est ce qu'il change pour un développeur classique ? Je présenterai aussi comment on peut déployer un cluster dans le cloud pour faire des essaies depuis de chez soi avec des offres gratuites.

par Horacio LASSEY
